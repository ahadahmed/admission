Applicant Dashboard

The following steps has to be followed by Applicant

  1. From Application menu, Applicant needs to verify his/her application

  2. From Payment menu, Applicant will get his/her Application ID and needs to pay application fee using SureCash

  3. From Profile menu, Applicant has to upload image

  4. From Enquiry menu, Applicant can submit any enquiry regarding admission

  5. From Admit Card menu, Applicant can download his/her admit card
